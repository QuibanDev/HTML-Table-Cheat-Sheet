HTML Table Cheat Sheet

Basic Table Structure

This is a basic table structure:

Basic Table Example
Header 1 Header 2 Header 3
Row 1, Cell 1 Row 1, Cell 2 Row 1, Cell 3
Row 2, Cell 1 Row 2, Cell 2 Row 2, Cell 3
Footer 1 Footer 2 Footer 3

Table Elements

<table>
Defines the table.
<thead>
Groups the header content in a table.
<tbody>
Groups the body content in a table.
<tfoot>
Groups the footer content in a table.
<tr>
Defines a row in the table.
<th>
Defines a header cell in the table.
<td>
Defines a standard cell in the table.
<caption>
Defines a table caption.

Attributes

  • colspan: Specifies the number of columns a cell should span.
  • rowspan: Specifies the number of rows a cell should span.
  • scope: Defines the scope of the header cell. Values can be col, row, colgroup, or rowgroup.
